# 速维电脑租赁管理系统 V2 — 部署指南

## 启动方式

### 方式 1：隐藏控制台窗口（推荐生产环境）
```bash
pythonw main.pyw
```
或直接在文件管理器中**双击 `main.pyw`** 文件启动

### 方式 2：显示控制台窗口（开发/调试）
```bash
python main.py
```

## 核心功能

### 已实现功能
- ✅ **用户认证**：多用户登录、角色权限管理、登录失败锁定机制
- ✅ **租赁管理**：租赁记录 CRUD、续租、状态跟踪、付款记录、合同导出
- ✅ **硬件管理**：品牌库管理、型号自动补全、参考定价
- ✅ **仪表板**：统计概览、到期提醒、快捷操作
- ✅ **报表分析**：统计图表（饼图/柱状图）、Excel 导出
- ✅ **批量操作**：批量续租、批量状态变更、批量导出
- ✅ **数据备份**：自动备份机制、手动备份入口

### 数据库
- 版本：DB_VERSION = 5
- 存储位置：`rental_data.json`
- 自动迁移：应用启动时自动检测并执行数据库升级

## 故障排查

### 黑框/闪烁问题
- 确保使用 `main.pyw` 启动（已解决）

### 数据库错误
- 检查 `rental_data.json` 文件权限
- 清除该文件可重新初始化数据库

### 硬件型号不显示
- 确认 `modules/hardware_models.py` 存在且包含型号数据
- 重启应用刷新型号缓存
