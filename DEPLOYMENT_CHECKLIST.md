## 问题处理

### 如果部署失败

1. **立即停止** - 停止部署流程
2. **评估** - 确定失败原因
3. **备份** - 检查备份完整性
4. **回滚** - 如需要执行回滚
5. **调查** - 进行根本原因分析
6. **修复** - 修复问题
7. **重试** - 再次尝试部署

### 回滚步骤

```bash
# 1. 停止当前应用
sudo systemctl stop suwei-rental.service

# 2. 恢复备份
tar -xzf /backups/suwei_rental_backup_YYYYMMDD_HHMMSS.tar.gz -C /opt/suwei_rental

# 3. 重启应用
sudo systemctl start suwei-rental.service

# 4. 验证
systemctl status suwei-rental.service
```
