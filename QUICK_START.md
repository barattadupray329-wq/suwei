## 🎯 5 步快速开始

### 步骤 1：主电脑启动（开发机器）

```bash
python main.py
```

或使用 exe：
```bash
dist\速维电脑租赁管理系统_v2.exe
```

输入用户名和密码登录

### 步骤 2：查看服务器 IP

程序会输出类似这样的日志：

```
✅ HTTP 同步服务器已启动: http://192.168.1.100:9999
```

**记住这个 IP**（例如：`192.168.1.100`）

### 步骤 3：浏览器验证（可选）

打开浏览器访问：
```
http://192.168.1.100:9999
```

你会看到绿色的 ✅ 状态页面

### 步骤 4：其他电脑配置

在其他电脑上运行：

```bash
python client_config.py
```

按提示输入：
1. 主电脑的服务器地址（例如：`http://192.168.1.100:9999`）
2. 其他设置（按回车使用默认值）

### 步骤 5：测试连接

配置脚本会自动测试连接，你会看到：

```
✅ 连接成功！
   服务器状态: running
   更新包数量: 0
```

## 🐛 常见问题

### Q: 其他电脑连接失败怎么办？

A: 检查这三点：
1. IP 地址是否正确（看主电脑的日志）
2. 防火墙是否允许 9999 端口
3. 主电脑是否还在运行

### Q: 修改代码后多久能看到更新？

A: 
- 主电脑：自动检测（更新通知每 30 秒检查一次）
- 其他电脑：每 60 秒检查一次，有新更新立即下载

### Q: 可以改服务器端口吗？

A: 可以，编辑 `modules/sync_server_manager.py` 的第 20 行：
```python
def __init__(self, project_root: str, port: int = 9999):  # ← 改这里
```

### Q: 主电脑关了会怎样？

A: 其他电脑无法下载更新，但之前已下载的更新可以继续用。
