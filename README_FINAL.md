## 📊 使用示例

### 1. 新建租赁
1. 点击"租赁管理" → "新增租赁"
2. 填写客户信息、硬件配置
3. 硬件型号支持自动补全（输入品牌/型号）
4. 点击"提交"保存

### 2. 记录付款
1. 在租赁详情中点击"收款"按钮
2. 输入付款金额、支付方式
3. 系统自动更新"已付金额"与"剩余金额"
4. 付款历史自动记录

### 3. 续租操作
1. 点击"续租"按钮
2. 系统自动生成新订单
3. 原订单标记为"已完成"

### 4. 查看统计报表
1. 点击"报表"菜单
2. 选择图表类型（饼图/柱状图等）
3. 可导出为 PNG 或 Excel

### 5. 批量操作
1. 在列表中使用 Ctrl+Click 多选记录
2. 点击"批量操作"
3. 选择操作类型（续租/状态变更/导出）

## 📞 技术支持

### 常见问题

**Q: 启动时出现黑框？**  
A: 使用 `pythonw main.pyw` 而非 `python main.py`

**Q: 数据无法保存？**  
A: 检查 `rental_data.json` 文件权限，或删除后重新初始化

**Q: 硬件型号不显示？**  
A: 确保 `modules/hardware_models.py` 存在，重启应用刷新缓存

**Q: 如何导出所有数据？**  
A: 在租赁列表中使用"批量导出"功能，或直接复制 `rental_data.json`

## 🎓 开发者指南

### 添加新功能
1. 在 `modules/` 中创建新模块
2. 实现模块的主 Frame 类（继承 tk.Frame）
3. 在 `core/app.py` 中注册导航项
4. 在主窗口添加对应的导入

### 数据库迁移
1. 编辑 `core/data_manager.py` 中的迁移函数
2. 增加 `DB_VERSION` 版本号
3. 启动应用自动执行迁移

### 打包为独立可执行文件
```bash
pip install pyinstaller
pyinstaller --windowed --add-data "theme:theme" --add-data "modules:modules" --add-data "core:core" main.pyw
```
