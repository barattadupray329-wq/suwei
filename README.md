# 💻 速维电脑租赁管理系统 V2 (Pro)

> 全新重构的深色主题租赁管理解决方案，集成 AI 智能助手与自动化数据管理。

## ✨ 功能特性

- 🎨 **深色现代 UI**：采用 DarkTheme 配色方案，视觉舒适，专业美观。
- 🤖 **AI 租赁助手**：
  - 📝 **智能提取**：从自然语言中自动识别姓名、电话、身份证、日期、金额等 9 类信息。
  - 💬 **即时问答**：内置合同/租金/逾期等常见业务问题库。
  - 🔍 **数据验证**：自动检查数据完整性与格式正确性。
- 📊 **全景仪表板**：实时展示租赁状态统计（在租、逾期、退租等）。
- 📋 **租赁全生命周期管理**：新增、编辑、删除、续租（自动计算到期日）。
- 📥📤 **Excel 兼容导入导出**：支持 UTF-8-sig 编码的 CSV 批量操作。
- 💾 **零依赖部署**：纯 Python 标准库实现，无需数据库，JSON 本地持久化。

## 📦 安装与运行

### 方式一：源码运行 (开发者)

1. **克隆或下载项目**
   确保文件结构如下：
   ```text
   速维电脑租赁管理系统_v2/
   ├── main.py              # 启动入口
   ├── core/                # 核心逻辑
   ├── modules/             # UI模块
   ├── theme/               # 主题配色
   └── data/                # 数据存储 (自动生成)
   ```

2. **运行主程序**
   打开终端（PowerShell/CMD），进入项目目录并执行：
   ```powershell
   python main.py
   ```

### 方式二：独立 EXE 运行 (用户)

如果您下载的是编译好的 `.exe` 文件：

1. 双击 `速维电脑租赁管理系统V2.exe`。
2. 系统会自动在当前目录创建 `data/` 文件夹用于存储数据。
3. **注意**：请勿将 exe 放置在只读目录（如 C:\Program Files）以免数据无法保存，建议放在 D 盘或桌面。

## 🔑 默认配置

- **管理员账号**: `admin`
- **管理员密码**: `admin123`

### 修改默认密码

1. 打开 `core/data_manager.py`。
2. 找到 `_get_default_data` 方法。
3. 修改 `default_admin` 和 `default_password` 的值：
   ```python
   def _get_default_data(self) -> Dict:
       return {
           "rental_records": [],
           "settings": {
               "default_admin": "your_new_name",      # 新用户名
               "default_password": "your_new_pass",   # 新密码
               "last_backup": None
           }
       }
   ```
4. 删除旧有的 `data/rental_data.json` 文件以重置配置。

## 📖 使用指南

### 1. 登录系统
启动软件后，输入账号密码点击"🚀 登录"。错误会有弹窗提示。

### 2. 导航菜单
- **左侧边栏**：
  - 📊 **仪表板**：查看数据概览（总记录、在租数、逾期数等）。
  - 📋 **租赁管理**：核心业务操作区。
  - 🚪 **退出系统**：安全关闭应用。

### 3. 租赁管理
- **搜索与筛选**：顶部搜索栏支持姓名/电话/ID 模糊搜索；下拉框支持按状态筛选。
- **续租操作**：选中记录点击"🔄 续租"，输入时间与金额，系统自动计算新到期日并更新总租金。
- **CSV 导入导出**：
  - **导出**：点击"📤 批量导出"，生成 Excel 可读的 CSV 文件。
  - **导入**：点击"📥 批量导入"，选择 CSV 文件，系统自动校验并去重。

### 4. AI 助手
点击租赁管理界面的 **🤖 AI 助手** 按钮打开面板：
- **自动填充**：输入如 `"租赁人张三，电话13800000000..."`，点击提取即可结构化数据。
- **智能问答**：点击预设问题按钮快速查阅业务规则。
- **信息提取**：将杂乱文本转化为标准 JSON 格式。

## 💾 数据管理

- **存储位置**: `data/rental_data.json`
- **备份机制**: 
  系统内置 `DataManager.backup_data()` 方法，可生成带时间戳的 JSON 备份文件（如 `data/backup_20260611.json`）。
- **数据恢复**: 若数据损坏，可手动删除 `rental_data.json`，重启软件将自动重建空数据库。

## ❓ 常见问题 (FAQ)

**Q: 打开软件闪退怎么办？**
A: 通常是因为杀毒软件拦截或路径包含特殊字符。请尝试以管理员身份运行，或将软件移动到纯英文路径下测试。

**Q: CSV 导入乱码？**
A: 本系统使用 `UTF-8-sig` 编码导出和导入。请确保您的 CSV 文件也是此编码（Excel 默认保存可能为 ANSI，建议在 Excel 中"另存为 -> CSV UTF-8"）。

**Q: AI 提取不准确？**
A: AI 模块基于正则表达式匹配。请尽量使用标准的自然语言描述（例如包含"租赁人"、"电话"、"起租"等关键词）。
